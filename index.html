Projects

Library Management System

The Library Management System project involves creating a comprehensive software solution to manage the operations of a library.This is developed using javaFX in eclipse IDE. It aims to streamline the processes of book cataloging, borrowing and returning books, managing user accounts, and generating reports. The system will provide an efficient and user-friendly interface for librarians and library patrons to interact with.

Online Marketplace

The Online Marketplace project aims to create a web-based platform where users can buy and sell various products. It provides a marketplace for sellers to showcase their products and buyers to browse, search, and purchase items. The application facilitates secure transactions, order management, and user ratings/reviews.It will utilize the MERN stack, which consists of MongoDB, Express.js, React, and Node.js.

Recipe Sharing Application

The Recipe Sharing Application is a web-based platform that allows users to discover, share, and save their favorite recipes. The application provides a user-friendly interface for browsing recipes, creating and managing personal recipe collections, and interacting with a community of food enthusiasts. It aims to simplify the process of finding and organizing recipes, making it easier for users to explore new culinary creations. It will utilize the MERN stack, which consists of MongoDB, Express.js, React, and Node.js.
